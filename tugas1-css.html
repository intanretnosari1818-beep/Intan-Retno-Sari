Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa style sheet yang digunakan untuk mengatur tampilan, tata letak, warna, font, dan dekorasi elemen HTML. CSS memisahkan konten (HTML) dari desain visual, sehingga mempermudah pembuatan situs yang responsif dan konsisten di berbagai perangkat.
